# Xgenium
A collection of opensource trading strategies; burnt onto xgenium nodes consisting of raspberry pi's and SD cards to create a microcomputer for customers to plug in and solve the common issue of staring at charts all day. 

Change frquency of signals 1d to 1h or 15m for faster signals. 

Volatility can be adjusted on lines 59 and 61, change to higher numbers for more volatile tickers.

Change the asset by using appropriate Yahoo finance tickers in the TICKERS list
